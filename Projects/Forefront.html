Introduction

Forefront is an intense VR shooter featuring 32-player large-scale battles, land, sea, and air vehicles, and immersive, tactical combat. You are the tip of the spear in the war between a global energy giant and resistance forces.

Technical Details

Engine: Unity

Multiplayer Solution: Photon Fusion 2

Platform: Meta, Steam, Pico (PC and Android)

My Role

  • Prototype:
    • Implementing and testing new gameplay ideas, closely working with the designers whilst iterating and improving these features, according to feedback.
    • Work on a system that closely resembles the unity ECS system. This allowed us to control the order of execution of everything in the game and allowed us to optimize a lot of basic functionality. This was later expanded in production and used in the final product.
  • Production:
    • Work on the first version of the bots, using the unity jobs system to update their vision in an optimized way, even more optimized by using spatial hashing to filter out unneeded calculations
    • Lead development on anything weapons related, from base posing and handling to supporting unique reloads while still using 1 generalized system and flow. Supporting multiple attachments per gun easily set up with a 1 click editor tool. This is all networked in an optimized way so it still functions when 32 players are present in the server
    • Easy-Anticheat Implementation
    • General gameplay programming implementations of new features.
